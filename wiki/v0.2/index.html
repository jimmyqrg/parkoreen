Parkoreen v0.2

Introduced the plugin system — Hollow Knight, HP, and Code plugins — plus improved portal visuals and mobile support.

What's new in v0.2

  • Plugin system — extensible plugin framework with HK, HP, and Code plugins
  • HK (Hollow Knight) plugin — soul statue, Mantis Claw, attack, pogo
  • HP plugin — health bar, damage/heal mechanics
  • Code plugin (Beta) — Python scripting via Skulpt
  • Improved portal and spike visuals
  • Load bar for asset loading
  • Mobile / touchscreen support improvements
This article documents the complete feature set of v0.2. For a diff from v0.1, see the CHANGELOG.

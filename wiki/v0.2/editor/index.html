✎ Editor

The map editor: placing, selecting, and deleting objects on a grid.

Overview

The editor works on a 32×32 px grid. Objects snap to grid cells. The camera can be panned with Middle-click drag or arrow keys, and zoomed with the scroll wheel.

Tools

ToolShortcutDescription
PlaceQClick/drag to place the selected object type.
EraseWClick/drag to delete objects.
Move CameraEPan the camera without placing.

Selection

Click an object to select it and open its config panel. Clicking empty space deselects.

Keyboard shortcuts

KeyAction
QSwitch to Place tool
WSwitch to Erase tool
ESwitch to Move Camera tool
DeleteDelete selected object
EscDeselect / close panel

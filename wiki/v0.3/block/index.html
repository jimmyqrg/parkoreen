v0.3

■ Block

Solid platform tiles — now with brick texture support.

Overview

Blocks are the primary solid tiles. v0.3 adds optional brick texture rendering.

Textures (new in v0.3)

Select a texture from the block config panel. Available in v0.3:

  • None — solid color fill.
  • Brick — tiled brick pattern overlay.

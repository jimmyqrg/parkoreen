Tools

ToolShortcutDescription
PlaceQClick/drag to place the selected object type.

Multi-selection (new in v0.3)

With the Select tool, drag a rectangle to select multiple objects. The Object Inspector panel shows properties shared by all selected objects and allows bulk editing.

Rotate Left

A new Rotate Left button appears in the toolbar when an object is selected. It rotates the object's visual orientation 90° counter-clockwise.

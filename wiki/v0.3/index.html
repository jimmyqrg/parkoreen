Parkoreen v0.3

Added saw blades, buttons, block textures, multi-selection, and cloud color settings.

What's new in v0.3

  • Saw Blade — spinning circular hazard (spinner)
  • Buttons — click-triggered interactive objects
  • Block textures — brick texture option
  • Multi-selection — select and configure multiple objects at once
  • Object Inspector — bulk property editing for selected objects
  • Rotate Left — new editor toolbar control
  • Heal at Checkpoints — option to restore HP on checkpoint touch
  • Cloud color settings — sky color and galaxy cloud color
  • UI transition improvements and stacking fixes
  • Variable mechanic block type
This article documents the complete feature set of v0.3. For a diff, see the Changelog.

🔧 Placement Toolbar

All options available on the side/bottom toolbar when placing blocks and spikes in v0.0.

Overview

When you select a placement mode (Block, Obstacle, or Spawn/End), a contextual toolbar appears. The options change depending on what you are placing.

Common options (all placement modes)

OptionTypeDescription
ColorColor pickerThe fill color of the object. Click the color swatch to open the color picker.
OpacityNumber 0–100How transparent the object is. 100 = fully opaque, 0 = fully invisible. Default 100.

Block options

OptionDescription
AppearanceSelect Ground (solid block) as the visual type.
CollisionToggle whether the block is physically solid. Disabling collision lets the block be used purely as decoration.
LayerWhich depth layer (0–4) new blocks are placed on. 0 is behind most objects; 4 is in front of everything.

Obstacle (spike) options

OptionDescription
Spike TouchboxOverride the world-default spike touchbox mode for newly placed spikes. See Spike for mode descriptions.
Damage AmountHow much HP the spike removes on contact (default 1). Only relevant when a health plugin is active.
LayerDepth layer for newly placed spikes.

Spawn / End options

OptionDescription
TypeSwitch between placing the Spawn point (where players start) or the End goal (where they must reach to finish).
ColorColor for the spawn/end marker visual.
Only one spawn point and one end point can be active at a time. Placing a new spawn point replaces the old one.

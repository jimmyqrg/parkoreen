v0.4

✎ Editor

The map editor — now with undo/redo, a dedicated Spawn & End tool, and drag-resize for saw blades.

Tools

ToolShortcutDescription
PlaceQClick/drag to place the selected object type.
EraseWClick/drag to delete objects.
SelectEClick or drag to select objects.
Move CameraRPan the view.
Spawn & End—New dedicated tool for placing spawn/end markers.

Undo / Redo (new in v0.4)

Editor actions are recorded in a history stack. Use Ctrl+Z to undo and Ctrl+Y (or Ctrl+Shift+Z) to redo.

Drag-resize for saw blades

When a Saw Blade is selected, drag handles appear at the edges for resizing without opening the config panel.

⚙️ Map Config

All configurable map-level settings available in v0.1.

General

  • Map NameUntitledDisplay name for the map.
  • Author(empty)Creator credit shown on the map screen.
  • Backgroundskysky or galaxy. Changes the scrolling backdrop.

Physics

  • Gravity0.71Downward acceleration applied each tick.
  • Jump Force-13.2Vertical velocity applied when the player jumps (negative = upward).
  • Player Speed5Base horizontal movement speed (px/tick).
  • Max Jumps1Number of jumps allowed before landing.
  • Infinite JumpsOffIf on, the player can jump again immediately after each jump.
  • Additional Air JumpOffGrants one extra jump when airborne (double-jump style).
  • Die Line Y1000The Y position (in pixels) below which the player dies instantly.

Spike settings

  • Spike TouchboxnormalGlobal hitbox mode for all spikes: normal, full, tip, ground, flag, air.
  • Drop Hurt OnlyOffIf on, spikes only kill the player when falling from above (ground touchbox variant).

Default colors

  • Block Color#787878Default color applied to newly placed blocks.
  • Spike Color#c45a3fDefault color for newly placed spikes.
  • Portal Color#9b59b6Default color for newly placed teleportals.

Checkpoint colors New v0.1

  • Inactive Color#808080Color of an untouched checkpoint.
  • Touched Color#2196F3Color of a checkpoint the player has passed but is not the active spawn.
  • Active Color#4CAF50Color of the checkpoint currently set as the respawn point.

Cloud colors

  • Cloud Color (Sky)#ffffffCloud color when background is sky.
  • Cloud Color (Galaxy)#9382a8Cloud color when background is galaxy.

What is new in v0.1?

  • Zones — invisible rectangular regions that can display a name label
  • Teleportals — linked portal pairs that teleport the player when touched
  • Checkpoint — respawn markers with animated states (default / touched / active)
  • Font size setting — accessible in the Settings panel
  • Horizontal flip — flip any object horizontally via the object inspector
  • Larger position values for bigger maps
  • Show current object position in the editor UI
  • Improved touchscreen support

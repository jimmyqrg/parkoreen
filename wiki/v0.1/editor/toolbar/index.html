📦 Placement Toolbar

All placement options available in the Add toolbar for v0.1.

Block

  • Color#787878Fill color of the block.
  • Opacity100%Transparency from 1–100 (0 = invisible).
  • Layer0Draw order. Higher layers render on top.
  • CollisionOnWhether the block is solid.

Obstacle (Spike)

  • Color#c45a3fSpike fill color.
  • Opacity100%Transparency 1–100.
  • Layer0Draw order.

Game Item — Checkpoint New v0.1

  • Opacity100%Transparency.
  • Layer0Draw order.

Game Item — Zone New v0.1

  • Name(empty)Label displayed inside the zone rectangle in the editor.
  • Color#9b59b6Border / tint color used in the editor view.
  • Opacity100%Transparency of the zone's editor visualization.

Game Item — Teleportal New v0.1

  • Teleportal Name(required)Unique identifier used to link portals together.
  • Send To(empty)Comma-separated list of teleportal names to send the player to.
  • Color#9b59b6Portal glow color.
  • Opacity100%Opacity of the portal.

Spawn / End

  • Type—Spawn point (player start) or End point (level goal).
  • Layer0Draw order.

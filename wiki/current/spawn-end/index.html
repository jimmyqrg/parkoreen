🚩 Spawn & End

The map's starting point and level goal object.

Spawn Point

The spawn point is where the player appears when the map starts. Every map must have exactly one spawn point. If no checkpoint has been activated, the player also respawns here on death.

The spawn point is placed using Add → Spawn / End → Spawn Point. It appears as a flag icon in the editor.

End Point

The end point is the level goal object. When the player touches the end point, the level is considered complete and the end-of-level screen is shown.

Placed using Add → Spawn / End → End Point. Shown as a flag or star icon in the editor.

Requirements

  • A map must have exactly one spawn point. Without one, the player has no start position.
  • A map should have at least one end point so the level can be completed.
  • Both are listed under the Spawn / End placement mode in the Add menu.

▶ Tester

Play-test your map directly inside the editor without saving or publishing.

Starting & stopping

Click Test Game in the editor toolbar to compile and launch the map in tester mode. The player spawns at the map's spawn point. Press Stop or Escape to return to the editor.

Controls

KeyAction
A / DMove left / right
W / SpaceJump
RRespawn at active checkpoint (or spawn point if none)
EscapeExit tester and return to editor

Tester toolbar options

  • InvincibilityOffPrevents the player from dying. Useful for exploring a map without hazards killing you.
  • Show TouchboxesOffDraws the player's Ground and Hurt hitboxes and spike hitboxes as colored overlays.
  • Show FPSOffDisplays the current frames-per-second counter in the top corner of the screen. Only visible in tester mode.
Tester settings (invincibility, touchboxes, FPS) are remembered between test runs within the same editor session.

Limitations

  • Tester mode does not save progress or affect the published map.
  • Multiplayer is not active in tester mode — you play as a single local player.
  • Plugin scripts run in the tester, so you can test Code plugin behaviors.

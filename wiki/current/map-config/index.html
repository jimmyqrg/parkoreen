⚙️ Map Config

All map-level settings in the current version of Parkoreen.

General

  • Map NameUntitledDisplay name for the map.
  • Author(empty)Creator credit shown on the map screen.
  • Backgroundskysky — daytime clouds. galaxy — space backdrop with purple clouds.

Physics

  • Gravity0.71Downward acceleration applied each tick. Higher = heavier feel.
  • Jump Force-13.2Vertical velocity on jump (negative = upward). More negative = higher jump.
  • Player Speed5Horizontal movement speed in pixels per tick.
  • Max Jumps1How many times the player can jump before landing.
  • Infinite JumpsOffIf on, jump count is never exhausted.
  • Additional Air JumpOffGrants an extra mid-air jump (double-jump).
  • Die Line Y1000Y position (px) at which the player instantly dies if they fall below it.

Spike settings

  • Spike TouchboxnormalGlobal hitbox mode: normal, full, tip, ground, flag, air, all-spike.
  • Drop Hurt OnlyOffSpikes only kill when the player is falling onto them.

Default colors

  • Block Color#787878Default color for new blocks.
  • Spike Color#c45a3fDefault color for new spikes.
  • Text Color#000000Default color for new text objects.
  • Portal Color#9b59b6Default color for new teleportals.
  • Bouncer Color#461A0CDefault color for new bouncers.

Coin settings

  • Show Coin CounterOnDisplays the coin counter HUD during gameplay when coins are present on the map.

Checkpoint colors

  • Inactive Color#808080Untouched checkpoint color.
  • Touched Color#2196F3Previously-passed checkpoint color.
  • Active Color#4CAF50Current active respawn checkpoint color.

Cloud colors

  • Cloud Color (Sky)#ffffffCloud tint when background is sky.
  • Cloud Color (Galaxy)#9382a8Cloud tint when background is galaxy.

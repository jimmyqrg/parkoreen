🖊 Editor

All editor tools, multi-select, keyboard shortcuts, and workflow in the current version.

Grid & snapping

All objects snap to a 32×32 px grid. Grid lines are visible on the canvas in edit mode.

Editor tools

RotateRClick an object to rotate it 90° clockwise.
Rotate Left—Click an object to rotate it 90° counter-clockwise.

SelectSDrag a selection rectangle to multi-select objects. See below.

Multi-select

Use the Select tool to drag a box around multiple objects. When objects are selected:

  • A combined Object Config panel shows shared properties (e.g. color if all selected objects share a color, opacity, spin direction for saw blades, etc.).
  • The Rotate, Duplicate, Erase, and Move actions apply to the whole selection.
  • Click an empty area to deselect.

Object inspector

Click any placed object to open its Object Inspector — shows all per-object config settings. For multi-selected objects, shared settings are shown and edited collectively.

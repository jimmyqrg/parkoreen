📦 Placement Toolbar

Every option available in the Add toolbar for the current version.

Block

  • Color#787878Fill color of the block.
  • TexturenoneOptional visual texture overlay: none, solid, brick.
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.
  • CollisionOnToggle collision.

Obstacle — Spike

  • Color#c45a3fSpike fill color.
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Obstacle — Saw Blade

  • Color#787878Blade fill color.
  • Spin Speed2Rotational speed in degrees per tick.
  • Spin DirectionClockwiseClockwise (1) or Counter-clockwise (-1).
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Game Item — Checkpoint

  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Game Item — Zone

  • Name(empty)Zone identifier label in editor.
  • Color#9b59b6Editor tint color.
  • Opacity100%Editor visualization opacity.

Game Item — Teleportal

  • Teleportal Name(required)Unique ID for linking portals.
  • Send To(empty)Comma-separated destination portal names.
  • Color#9b59b6Glow color.
  • Opacity100%Transparency 0–100.

Game Item — Coin

  • Coin Amount1How many coins this pickup awards.
  • Coin Activity Scopeglobalglobal — collected once for everyone. player — each player collects independently.
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Game Item — Bouncer

  • Bouncer Strength15Launch velocity; range 5–50.
  • Bouncer Direction0° (up)Launch angle: 0 = up, 90 = right, 180 = down, 270 = left.
  • Color#461A0CBouncer pad color.
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Game Item — Button

  • Button Interactionclickclick — player clicks the button. collide — activates when the player walks onto it (pressure plate style).
  • Only OnceOffIf on, the button can only be activated once per session.
  • Display Name(empty)Label shown on or above the button.
  • Display Description(empty)Tooltip description shown when hovering the button.
  • ColorautoButton body color.
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Text

  • Content(empty)The text string displayed in the world.
  • FontParkoreen GameFont family. Built-in: Parkoreen Game. Additional fonts may be added via the Code plugin.
  • Font Size16Text size in pixels.
  • Color#000000Text fill color.
  • Horizontal Alignleftleft, center, or right.
  • Vertical Aligntoptop, center, or bottom.
  • Horizontal Spacing0Extra letter spacing in pixels.
  • Vertical Spacing0Extra line spacing in pixels.
  • Opacity100%Transparency 0–100.
  • Layer0Draw order.

Spawn / End

  • Type—Spawn point (player start) or End point (level goal).
  • Layer0Draw order.

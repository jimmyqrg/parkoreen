🟣 Teleportal

Linked portals that instantly transport the player between map locations.

Overview

Teleportals are portal objects that warp the player to a named destination portal when touched. They emit a particle ring effect. Portals are linked by matching their teleportalName identifiers in the Send To field.

Linking portals

  1. Place portal A — set name to A, set "Send To" to B.
  2. Place portal B — set name to B, set "Send To" to A for two-way travel.

Entering portal A teleports the player to portal B's position, and vice versa. You can link one portal to multiple destinations by comma-separating names in the Send To field.

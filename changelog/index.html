Changelog

All notable changes to Parkoreen, newest first. Current is the unreleased tip on next.

Current dev unreleased Ongoing · branch next
Admin & Moderation
  • Admin Drag Player — press M to toggle Quick Drag Player (joins G for Fly and K for Quick Kill).
  • Admin panel: users, rooms, and maps tab; scrolling support; access and layout fixes.
  • Admin chat commands in-game; Parkoreen admin welcome flow.
  • Maps: list/detail, rename/delete, room↔map linkage; admin edit/host flows.
  • Reserved official names handling; role-based admin powers.
Audio
  • New sounds — bounce.ogg on bouncer trigger; button.ogg on button activate; checkpoint.ogg on first checkpoint touch; endpoint.ogg on reaching the end.
  • Jump sound — switched from mp3/jump.mp3 to ogg/jump.ogg.
  • Place & erase sounds — tile.ogg plays when any object is placed in the editor; erase.ogg plays when objects are erased.
Editor & Maps
  • Coin config — Amount field (value added to counter, default 1) and Activity Scope toggle (Global — coin changes for all players; Player — only the collector sees the change). Color picker added (default #FFDD00).
  • Bouncer config — direction picker with rotating sprite preview (up / right / down / left). Match Appearance toggle controls whether launch direction and visual orientation are linked. Default color updated to #461A0C.
  • Button config — One-Time Trigger option (fires only once per session). Two color pickers: Face Color (default #F52C2C) and Base/Border Color (default #CFCFCF).
  • Default colors updated to match SVG assets: bouncer → #461A0C; button face → #F52C2C; button base → #CFCFCF; coin → #FFDD00.
  • Bouncer spring animation — pad extends on trigger, compresses hard, then small oscillations before settling. Amplitude ramps up (max ×3.5×) on rapid re-trigger.
  • Coins — collectible game item. Float up/down, disappear on contact, play coin.ogg. Coin counter shown during tester/host mode; controlled by Show Coin Counter in map config; hidden automatically if the map has no coins.
  • Button interaction types — Click (popup on entry, confirms before triggering) and Collide (triggers silently on contact, renders as a pressure plate). Serialized as buttonInteraction.
  • Default portal / bouncer colors in map config — defaultPortalColor (default #9b59b6) and defaultBouncerColor (default #f59e0b); new objects seed from these world defaults.
  • Bouncer — new game item that launches the player upward; configurable strength (5–50); spring-pad visual; 200 ms cooldown.
  • Tester mode — press R to respawn at spawn point / last checkpoint.
  • Host game bug fix — Editor.prototype.hostGame override moved inside the DOMContentLoaded callback so clicking Host Game and ?mode=host links work correctly.
  • Auto-save on unload — beforeunload fires a keepalive fetch to save the map when the tab is closed or reloaded.
  • Undo / redo for editor actions.
  • Spawn & end markers — dedicated add-menu tool; "Koreen" label → Game Item.
  • Saw blade drag-resize — W/H inputs replaced with an eight-handle drag system. Click Adjust Size in the inspector to enter resize mode; press Escape or click Stop Adjusting to exit.
  • Saw blades — size unit controls, spin direction, damage amount; updated artwork.
  • Damage amount on spikes/saw blades; touchbox tuning (gaps, teleportal size, tester tool, invincibility testing).
  • Sample map updates; download map from dashboard; various default value tweaks.
Gameplay & Plugins
  • All Spike touchbox — new spike mode where the flat base is also lethal (no safe standing zone).
  • Bouncer in editor mode — applies launch force to the player while editing (not just during test/play).
  • FPS counter — now only visible in tester mode.
  • Touchbox memory — touchboxes are hidden when exiting tester but remembered; re-entering tester restores them.
  • Hollow Knight plugin — attack range/speed, controllable jump, keyboard fixes; CJ plugin added.
  • Teleportals — particle effects, collision/touch fixes.
  • HP bar appearance; HP-related fixes; pogo / movement fixes.
  • Performance — SpatialHash deduplication uses integer stamps (replaces string-key Set); circle particles batched per color+alpha group; portal particles skip redundant globalAlpha writes; off-screen particles culled from the update loop; solid block highlights/shadows use two batched beginPath calls instead of four fillRect calls.
  • Fly movement and performance improvements.
  • Lower max fall speed; text position and color-picker fixes.
Clouds & Backgrounds
  • More cloud SVG variants and loader updates; cloud generation behavior changes.
  • Grid and cloud rendering fixes.
Account & UI
  • Auto log-out when session is unauthorized.
  • Keyboard settings moved into Parkoreen settings; dashboard scrolling.
  • Editor (editor.js) updates.

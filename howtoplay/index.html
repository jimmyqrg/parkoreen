help How To Play

Learn how to play Parkoreen and master the parkour!

sports_esports Basic Gameplay

Parkoreen is a 2D platformer parkour game. Your goal is to navigate through obstacle courses and reach the end point!

lightbulb Game Objective
Start at the spawn point, navigate through the map, touch checkpoints to save your progress, and reach the end point to complete the level!

keyboard Controls

Action Keyboard Touchscreen
Move Left A or ← Joystick Left
Move Right D or → Joystick Right
Jump W ↑ or Space Jump Button
flight Fly Mode (Editor/Testing)
In the editor or test mode, press G to toggle fly mode. This lets you move freely in all directions to explore the map.

category Game Objects

square

Blocks

Solid platforms you can stand on and jump from. These form the main structure of parkour courses.

warning

Spikes

Dangerous obstacles! Touching the pointed part will kill you and send you back to the last checkpoint. The flat base is usually safe to stand on.

flag

Checkpoints

Touch these to save your progress! If you die, you'll respawn at the last checkpoint you touched. They change color when activated.

play_arrow

Spawn Point

Where you start the game. This is your initial respawn location until you touch a checkpoint.

emoji_events

End Point

The finish line! Reach this to complete the level and see your completion time.

move

Teleportals

Magical portals that teleport you to another location! Walk into one to be instantly transported to its connected portal.

dangerous Understanding Spikes

Spikes can have different behaviors depending on the map settings:

Normal Spike: The flat base acts as ground (safe to stand on), but the pointed parts will hurt you.
Full Spike: The entire spike is dangerous - don't touch any part of it!
Tip Spike: Only the very tip is dangerous. The base is ground, and the sides are safe.
info Drop Hurt Only
Some maps have "Drop Hurt Only" enabled. This means spikes only hurt you when you're moving toward the spike's tip. For example, a spike pointing up won't hurt you if you jump up through it - only if you fall down onto it!

group Multiplayer

Play with friends in real-time!

add Joining a Game
  1. Click Join on the Dashboard
  2. Enter the 6-character room code
  3. Enter password if required
  4. Start racing!
broadcast_on_personal Hosting a Game
  1. Open your map in the editor
  2. Click Config → Host Game
  3. Set max players and optional password
  4. Share the room code with friends!

settings In-Game Settings

Access settings anytime by clicking the ⚙️ Settings button (top-right in editor/play mode):

volume_up Volume
Adjust the game audio volume from 0% to 100%.
text_fields Font Size
Scale all text in the game from 50% to 150%. Useful for accessibility or different screen sizes.
touch_app Touchscreen Mode
Enable touch controls for mobile devices. Shows a joystick and jump button on screen.
home Back to Dashboard
Return to your map list at any time. You'll be asked to confirm before leaving.

tips_and_updates Pro Tips

speed Momentum
Keep moving! Your character maintains momentum, so running jumps go further than standing jumps.
replay Checkpoint Reset
Standing on a checkpoint and quickly switching directions (left-right or right-left) will reset your jumps! Useful for tricky sections.
zoom_in Zooming
Use Ctrl + Scroll to zoom in/out and see more of the map. Use Ctrl + Shift + Scroll to reset zoom.
